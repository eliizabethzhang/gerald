Gerald Help: Command List

.class create [name] : create class, give the class's list a name
.class join [code] : join a class
.list show [list] : list all your tasks from just specified list, or from all if no list name given
.list add [task] [list] : add task
.list finish [task] : Command to check off work (user side only)
.list edit [task] : Command to adjust due dates/other attributes of work item
.remind [list][task] : Command to create a reminder event (for studying for tests and stuff)

tip!

Have a big task that you want to break into subtasks? Instead of list, use bigtask!
ex: .bigtask add [task] [list]
And if you want to add a subtask to a bigtask, use a bigtask name in place of a list name!
ex: .bigtask add [task] [bigtask]
